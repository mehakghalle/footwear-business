# 👟 Footwear Business Website

A full-stack e-commerce footwear website built using React and Django REST Framework.

## 🚀 Features

- Men, Women, Kids sections
- Product filtering & search
- Add to cart system
- Checkout system
- Order stored in database
- Admin panel for product management
- Responsive UI

## 🛠 Tech Stack

Frontend: React  
Backend: Django + DRF  
Database: SQLite  

## ▶ Run Backend

```bash
cd backend
python manage.py runserver
